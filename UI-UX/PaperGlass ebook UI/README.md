## 📱 Overview
This capstone project for the *UI/UX Wireframing and Prototyping* certification (Coursera/SkillUp) establishes a seamless, user-centered reading platform. The design successfully adapts a complex 12-column desktop grid into a streamlined 4-column native mobile experience.

## ✨ System Architecture
* **Responsive Layouts:** Desktop (1440px) and Mobile (393px) views optimized for readability.
* **Component Library:** Auto-layout UI elements with mapped interactive states (Hover, Pressed, Active).
* **Interactive Prototyping:** Engineered user flows, including a custom slide-in mobile navigation menu overlay.
* **Accessibility:** 8pt grid system spacing and WCAG-compliant 4.5:1 contrast ratios.
